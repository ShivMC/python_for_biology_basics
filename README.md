# Python for Bioinformatics Workshop

This repository contains all codes, datasets, slides, and assignments for the Basic Python and Bioinformatics Workshop.

## Topics Covered
- Python Basics
- Data Types and Loops
- File Handling
- Working with FASTA Files
- GC Content Calculation
- Biopython
- Sequence Translation
- Sequence Alignment Basics

## Requirements
- Python 3.x
- Biopython

Install Biopython:
pip install biopython
